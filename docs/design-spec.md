# 语声 - 设计规范

## 设计原则

1. **简洁直观**：界面简洁，功能一目了然
2. **反馈及时**：每个操作都有即时反馈
3. **容错设计**：错误信息友好，提供恢复方案
4. **一致性**：跨平台UI风格统一

## 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #2196F3 | 蓝色，用于按钮、强调元素 |
| 辅助色 | #4CAF50 | 绿色，用于成功状态 |
| 警告色 | #FFC107 | 黄色，用于警告提示 |
| 错误色 | #F44336 | 红色，用于错误提示 |
| 背景色 | #FAFAFA | 浅灰，主背景 |
| 文字色 | #212121 | 深灰，主要文字 |
| 次要文字 | #757575 | 灰色，辅助文字 |

## 图标规范

- 使用Material Icons图标库
- 图标大小：24px（标准）、48px（大图标）
- 图标颜色与文字色保持一致

## 字体规范

| 用途 | 字体 | 大小 |
|------|------|------|
| 标题 | 系统默认中文字体 | 20px Bold |
| 正文 | 系统默认中文字体 | 14px Regular |
| 辅助文字 | 系统默认中文字体 | 12px Regular |
| 代码/数字 | 等宽字体 | 14px |

## 布局规范

- 间距：8px基础网格系统
- 边距：16px标准边距
- 圆角：4px（小元素）、8px（卡片）、12px（对话框）

## 组件规范

### 语音输入按钮
- 大圆形按钮，直径64px
- 录音中显示波纹动画
- 状态色：默认蓝色、录音中红色、处理中黄色

### 状态指示器
- 绿色圆点：就绪
- 红色脉冲：录音中
- 黄色旋转：处理中
- 灰色：离线

### 设置面板
- 左侧导航，右侧内容
- 分组设置项
- 底部保存/取消按钮

## 动画规范

- 过渡动画：200ms ease-in-out
- 按钮点击：100ms 缩放动画
- 状态切换：300ms 渐变
- 列表滚动：平滑滚动

## 无障碍设计

- 支持高对比度模式
- 按钮最小触摸区域：48x48px
- 提供完整的键盘导航
- 文字大小可调节
